This repository is aimed as a support and documentation for the submission of a research paper to MDPI.

Title: Physics-Informed Fuzzy Regression for Aeroacoustic Prediction Using Clustered TSK Systems

DOI:Drones 2026, 10(3), 200; [https://doi.org/10.3390/drones10030200](https://doi.org/10.3390/drones10030200)

This concatenate a research split between a dimensionnality reduction benchmark (preprint folder) linked to the publication found at : https://arxiv.org/abs/2505.23746. 
The second part is about integrating physical heuristics to counterbalance the loss of generality and informations brought in by the dimensionnality reduction.
